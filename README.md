# Product and Sales Data Analysis

## Overview
This project analyzes product and sales data to identify trends, performance patterns, and actionable business insights using Excel.

## Datasets
- Two Excel datasets containing product details, pricing, quantities sold, and revenue-related attributes.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Visual Analysis

## Key Objectives
- Analyze product-wise sales performance
- Identify revenue and quantity trends
- Highlight top-performing and underperforming products
- Support data-driven business decisions

## Deliverables
- Cleaned datasets
- Analytical report with insights and visualizations (PDF)

## Skills Demonstrated
- Exploratory Data Analysis (EDA)
- Business Analysis
- Data Interpretation
- Reporting & Documentation
